Exercise 2

Markdown Extra

Markdown Extra is a lightweight markup language based on Markdown implemented in PHP, Python and Ruby.
It adds features that are not available with plain Markdown syntax.

Markdown Extra adds following features

  • Markdown markup inside HTML blocks
  • Elements with id/class attribute
  • "Fences code blocks" that span multible lines of codes
  • Tables

public class Person{
  private string name;
  private int age;

  public Person(string initialName)
  {
    this.age = 0;
    this.name = initialName;
  }
  public void PrintPerson()
  {
    Console.WriteLine(this.name + ", age " + this.age + " years");
  }
  public void GrowOlder()
  {
    this.age = this.age + 1;
  }
}
